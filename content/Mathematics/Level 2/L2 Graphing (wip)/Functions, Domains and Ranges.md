## <u>Functions:</u>
A function is a relation in which there are no repetitions of x-values in the coordinate pairs; and yes, it is the $f(x)$ you see everywhere in maths, and you finally learn it.

For example:

<img src="/images/376f078bbfa5d31e205aa6792d314b7d.png" style='width: 300px;' />

The graph above is an example of a function, as there are only 1 $x$ value per corresponding $y$ value in their coordinate pairs. It also passes the vertical line test, which will be covered later in this note. 

A function must have only 1 pair of $x$ value for 1 pair of $y$ value (1 -> 1 ).

<img src="/images/2ce01aa5f5c5be0bb2c4e8acfafd01a1.png" style='width: 300px;' />

The graph above is not an example of a function, as there are multiple number of $y$ values in the same $x$ value in their coordinate pairs. On the graph, there are 2 $y$ values in the same pair of $x$ values.

To find out if a graph is a function or not, we use the vertical line test:

#### <u>Vertical Line Test (VLT):</u>

Passes VLT? -> Function

Doesn't Pass the VLT? -> Not a Function

<img src="/images/6b47f9576a9fca514574f0257ef47b1d.png" style='width: 500px;' /><img src="/images/bd48e7d17fe952c380a7270eb619adb9.png" style='width: 500px;' />

A vertical line testing is where you put multiple vertical lines on a graph to see if the graph intersects the same line twice (or more). As seen from the images, the graphs do not intersect the same line twice or more, meaning it is a function. 

Exception: if you do use a linear line in the exam, those won't be marked (even though it's a function)


<img src="/images/0491503530a80e4173802b16c23b8140.png" style='width: 500px;' /><img src="/images/976fa1204bfac459f2b9aee7bd7addcc.png" style='width: 500px;' />

These graphs fail the vertical line testing, as the graphs (orange and pink) intersects the same line twice, as well as all other lines. This means that this graph is not a function. 

However, if you restrict them, in terms of using domain & range, which will be covered later, they can indeed be a function as there will be no repeating $x/y$ values.

In very easier words, if you can express it in the form of $y=$, it is a function (most of the time)

**You have to use at least 4 functions in the exam (3 minimum, but 4 as safety). You may use linear lines but those will not be marked. You cannot use non-functions.**

## <u>Domains & Ranges</u>

Domains and ranges are used when you're restricting your functions, like, cutting them so that it doesn't go past a certain value. $0\leq x \leq 5$, $y>0$ might be something you probably saw on websites like Desmos, or any other resources. 


Restricting the $x$ value is restricting domain, and restricting the $y$ value is restricting the range.

<img src="/images/d45d8452f15060926d5a1370ce6059ee.png" style='width: 300px;' />

( $y^{2}=2x\ \left\{y\geq0\right\}$ )

In the function right above (same function without range is quite above as well), the range is restricted to '$y$ must be equal to or over 0', to make the graph a function. Using a vertical line test, we can see that this is a function as there are no repeating $x$ values, thus passing the VLT.

By restricting the domain/range so that it does not produce repeating $x$ values, we can make this a function.
