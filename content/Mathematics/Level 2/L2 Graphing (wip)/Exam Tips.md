#L2MAEL 

---

This exam is about either tracing, or making your own paths of an image/map using mathematical graphs, and explaining them in words.
Normally you are allocated up to three hours for this exam. I ran out of time myself (but I still got excellence L), because drawing them took ages.

### <u>Must talkabout(s) in the exam</u>:
You should talk about minimum three of these in the exam (go with four/five as safety).

1) Domain “$x$”
	The set of possible $x$-values which are valid for the context

2) Range “$y$” 
	The set of possible $y$-values which are valid for the context

3) Symmetry (if any)
	Describe any reflection lines i.e. give the equation of lines of symmetry

4) Turning/inflection points (vertex), local max/min points (if any)
	State any stationary point and give their coordinates

5) Intercepts (if any)
	Write down the values of the $x$ and $y$ intercepts.
	$y$-intercept at $(0,y)$ i.e. the value of $y$ when $x$ is 0.
	$x$-intercept(s) at $(x,0)$ i.e. the value of $x$ when $y$ is 0.
	*schools normally don't like it when the intercepts, or coordinates are like a never-ending decimal such as $(6.4869164511,10)$, and yes, I got flagged for it.*
	*when picking intercepts, try to use a good point (I refer to this as perfect points) like $(2,1)$.*

6) Asymptotes (if any)
	An asymptote is a line (value) that approaches a curve (equation) but doesn’t get to. 
	E.g. $x$ value causes the equation to divide by zero. If there are asymptotes, write down their equations.
