A generic square root function graph looks like this:
$$\text{equation:} \space\space y=\sqrt{ x }$$
<img src="/images/e853d07f90dbeabdb9bf761b008e4ae7.png" style='width: 300px;' />
..just like a sideways half parabola. 

The reason why it's a 'half' sideways parabola is because it is a positive square root. If it has a negative square root:
$$\text{equation:} \space\space y=-\sqrt{ x }$$
<img src="/images/dc68aff3003aa059e55ba52a3ec1e9a0.png" style='width: 300px;' />
..it's the direct opposite. 
This is because this function originates from the equation:
$$y^2=x$$
However, if we make y the subject: $y=\pm \sqrt{ x }$
If it has a $\pm$, it will not be a function as it fails the VLT ([[wndx2/2. Area/! school/Year 11/L2MAEL/Graphing - 91257/{wip} graphing/AS91257 - Functions & Domains]]).
And in L2 Graphing, we only use functions. Therefore, we only either use a $+$, or a $-$ for rooting so that it remains a function.

In this note, I will call the starting point of the graph the 'vertex'

This is the generic equation:
$$y=a\sqrt{ (x-p) }+q$$
..where $(p,q)$ is the coordinates for the start of this SQRT function. I call this te vertex/starting point.
This is one of the easiest graphs to use, as this is very straightforward.
